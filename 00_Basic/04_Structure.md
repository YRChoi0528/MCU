# 04. 구조체 (Structure)

## 목적
구조체를 “서로 관련된 데이터를 하나로 묶는 사용자 정의 자료형”으로 이해하고,
`.` / `->` 접근, 값 전달 vs 주소 전달, 배열과 결합되는 기본 패턴을 익힌다.

## 학습 내용
- `struct` 정의와 변수 선언
- 멤버 접근: `.` / `->`
- 구조체 값 전달(복사)과 포인터 전달(원본 수정)
- 구조체 배열과 반복 처리
- `.h`에 타입 선언을 두는 이유(여러 파일에서 같은 타입 공유)

## 파일 구성과 역할
- `04_structure.h`
  - 구조체 타입 선언과 함수 선언(원형)
- `04_structure.c`
  - 함수 정의(구현)와 예제(main)

## 핵심 정리
- 구조체는 “여러 값을 하나로 묶은 것”이다.
- 구조체를 함수로 넘기면 기본은 복사(값 전달)이며,
  원본을 바꾸려면 구조체 포인터를 넘긴다.
