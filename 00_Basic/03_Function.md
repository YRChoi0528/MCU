# 03. 함수 (Function)

## 목적
함수를 “코드를 묶는 단위”로 이해하고,
함수 호출 시 데이터가 어떻게 전달되는지(값 전달 / 주소 전달)를 구분한다.

## 학습 내용
- 함수 선언(원형)과 정의(본문)
- 반환값과 매개변수
- 값 전달: 함수 안에서 바꿔도 원본은 안 바뀜
- 주소 전달: 포인터를 넘기면 원본이 바뀜

## 핵심 정리
- “함수를 쓴다”는 것은 보통 **선언을 보고 호출**하고, 실제 동작은 **정의에 있다**는 뜻이다.
- 값 전달과 주소 전달 차이를 구분할 수 있어야 한다.
