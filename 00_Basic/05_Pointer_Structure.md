# 05. 구조체 포인터 (Pointer + Structure)

## 목적
구조체 변수 자체(`S s`)와 구조체 포인터(`S *p`)의 차이를 구분하고,
`.`와 `->`의 사용 조건을 정확히 이해한다.
또한 함수 인자로 구조체 포인터를 넘겨 원본을 변경하는 패턴을 익힌다.

## 학습 내용
- `S s` (구조체 변수) vs `S *p` (구조체 포인터)
- `s.a` : 구조체 변수의 멤버 접근
- `p->a` : 구조체 포인터가 가리키는 대상의 멤버 접근 (`(*p).a`와 동일)
- 함수로 `S`를 넘기면 복사(값 전달), `S*`를 넘기면 원본 변경 가능
- 구조체 포인터 배열처럼 쓰기: `p[i].a`, `p[i].b`
- 동적 할당 기본: `malloc`/`free` (필요한 만큼 구조체 공간 확보)

## 파일 구성과 역할
- `05_pointer_structure.h`
  - 타입 선언과 함수 선언(원형)
- `05_pointer_structure.c`
  - 함수 정의(구현)와 예제(main)

## 핵심 정리
- `.`는 “구조체 변수”에 사용
- `->`는 “구조체 포인터”에 사용
- 포인터를 넘기면 함수에서 원본을 바꿀 수 있다
