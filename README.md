# MCU

## 1. 과목 개요 및 실습 배경

본 리포지터리는 대학 학부 과정에서 수강하고, 이후 TA로 참여했던 **마이크로프로세서 실험 과목**의 실습 내용을 정리한 아카이브이다.

해당 과목은 MCU 기반 임베디드 시스템의 기본 구조를 이해하고, 다양한 주변장치를 직접 제어함으로써 **하드웨어와 소프트웨어 간의 연동 방식**을 학습하는 실습 중심 과목으로 구성되어 있다. 학기 말에는 Term Project를 통해 배운 내용을 종합적으로 적용하는 설계 과제를 수행하였다.

실습에서는 한백전자에서 제공한 MCU 키트를 사용하였으며, 공급사에서 제공한 `.c` / `.h` 파일을 기반으로 C 언어 환경에서 실습을 진행하였다. 학생들은 제공된 코드 구조를 분석한 후, 기능을 추가하거나 수정하는 방식으로 실습을 수행하였다.

---

## 2. 실습 환경 및 개발 흐름

### 2.1 사용 환경

- **MCU 키트**: HBE-MCU-Multi (한백전자)
- **개발 언어**: C
- **개발 환경**: IAR Embedded Workbench

### 2.2 빌드 및 다운로드 절차

1. IAR Embedded Workbench에서 프로젝트 빌드 (Project → Clean , Project → Make)
2. `.hex` 파일 생성
3. DFU File Manager를 이용해 `.hex` → `.dfu` 변환
4. Dfuse Demonstration을 통해 MCU 장비에 다운로드
5. 실제 하드웨어 동작 확인

이 과정을 반복적으로 수행함으로써, **빌드 산출물과 실제 하드웨어 동작이 연결되는 흐름**을 실습 환경에서 경험하였다.

---

## 3. 실습 수업 구성 내용

실습 수업은 주변장치 단위로 단계적으로 구성되었으며, 각 실습은 기능 이해 → 코드 구현 → 실제 하드웨어 동작 확인의 흐름으로 진행되었다.

### 3.1 GPIO
- Output / Input

### 3.2 Timer
- Timer Counter
- PWM, Piezo

### 3.3 Interrupt
- External Interrupt
- Timer Interrupt

### 3.4 UART
- Normal UART
- Interrupt UART

### 3.5 Peripheral Devices
- Step Motor
- Text LCD

### 3.6 ADC / DAC
- Photo Diode (ADC)
- Serial / Parallel (DAC)

### 3.7 SPI / I2C
- Serial Flash Memory (SPI)
- Temperature / Humidity Sensor Module (I2C)

각 실습은 단순한 코드 작성에 그치지 않고, **설정한 레지스터와 핀 동작이 실제 하드웨어에서 어떻게 반영되는지**를 확인하는 데 중점을 두었다.

---

## 4. TA 역할 및 수행 내용

본 과목에서 총 **4학기 동안 TA로 참여**하였으며, 주요 역할은 학생 코드 디버깅, 동작 원리 설명, 실습 진행 지원이었다.

### 4.1 학생 코드 디버깅

학생마다 코드 구조와 구현 방식이 달랐기 때문에, 개인적으로 작성한 표준 코드로 일괄 수정하지 않고 **학생이 작성한 코드 구조를 최대한 유지한 상태에서 오류를 수정**하였다. 이를 통해 학생들이 자신의 코드 흐름을 잃지 않도록 지원하였다.

### 4.2 코드와 하드웨어 동작의 연계 설명

- 레지스터 설정, 핀 설정, 인터럽트 설정이 실제 하드웨어 동작에 어떻게 반영되는지 설명
- 단순 문법 설명이 아닌, **"이 코드가 실행되면 하드웨어에서 무엇이 바뀌는가"**를 중심으로 안내

### 4.3 빠른 오류 분석 및 해결

제한된 실습 시간 내에 다수의 학생을 지원해야 했기 때문에, 오류 발생 시 **증상 → 원인 → 해결 방향**을 빠르게 판단하는 데 중점을 두었다. 특히 다음 항목을 우선적으로 점검하였다.

- 컴파일 오류
- 프로젝트 및 옵션 설정 누락
- 핀 설정 및 주변장치 초기화 오류

---

## 5. 리포지터리 목적

이 리포지터리는 단순한 코드 모음이 아니라,

- 마이크로프로세서 실험 과목의 구조 정리
- 실습 흐름과 하드웨어 동작의 연결 복기
- 임베디드 시스템 학습을 위한 참고 자료

를 목적으로 한다. 향후 임베디드 시스템 관련 학습 및 프로젝트를 진행할 때 기초 자료로 활용하는 것을 목표로 한다.
